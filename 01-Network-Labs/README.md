# Network Labs

This folder contains foundational networking and virtual infrastructure labs focused on IP addressing, subnetting, host-only network configuration, local routing behavior, and client-server communication inside VMware environments.

These labs establish the practical networking baseline required for later cybersecurity enumeration, scanning, and defensive analysis projects.
